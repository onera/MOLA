In this folder you will find several tutorials to start
learning how to use BEMT analysis and design functions.

The recommended order of execution is the following one:

(1) - BEMT_MinimalEx.py 

(2) - BEMT_TrimEx.py

(3) - BEMT_PropPolarEx.py

(4) - BEMT_MILdesignEx.py

(5) - BEMT_DirectDesignEx.py
